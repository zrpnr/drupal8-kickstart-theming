Defining Libraries

External


  angular.angularjs:
    remote: https://github.com/angular/angular.js
    version: 1.4.4
    license:
      name: MIT
      url: https://github.com/angular/angular.js/blob/master/LICENSE
      gpl-compatible: true
    js:
      https://ajax.googleapis.com/ajax/libs/angularjs/1.4.4/angular.min.js: { type: external, minified: true }
You can also define external librares that do not live in your theme. This is nice because you can take advantage of cdn hosted files, speeding up the time to load of your pages since this uses a non-blocking http request.
