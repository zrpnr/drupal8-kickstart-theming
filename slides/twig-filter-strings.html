Filter Strings


  {% set greeting = 'Hello Joe' %}

  {{ greeting }}               {# 'Hello Joe' #}  
  {{ greeting|t }}             {# 'Hola José' #}
  {{ greeting|upper }}         {# 'HELLO JOE' #}
  {{ greeting|t|upper }}       {# 'HOLA JOSÉ' #}
  {{ greeting|clean_class }}   {# 'hello-joe' #}

  {% filter upper %}
    Filter a whole bunch o text at once.
  {% endfilter %}
  
  {# 'FILTER A WHOLE BUNCH...' #}
There are a number of built in filters for twig, and a few that Drupal has added. You can uppercase a string, create a safe class identifier, or use Drupal specific filters like t which you'll remember if you're a D7 developer.
