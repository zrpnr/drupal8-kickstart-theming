  {% if label %}
    
{{ label }}

  {% endif %}

  {% if temperature > 50 and temperature < 90 %}
    
It's a nice day for a walk in the park.

  {% elseif temperature > 95 %}
    
It's a nice day for a swim.

  {% else %}
    
Take a nap. You deserve it.

  {% endif %}
Twig has limited template logic but you can use it to switch between printed values, just like how you'd be familiar with regular php templates. Note the syntax difference for logical tags- instead of {{ }} you have {% %}
