{{ loop.index0 }}    {# The current iteration of the loop. (0 indexed) #}

{{ loop.length }}    {# The number of items in the sequence #}
{{ loop.parent }}    {# The parent context #}
if you do a loop in twig, you have access to a special variable, named ...loop... this object has its own properties such as the current iteration fo the loop, either starting with 1 or 0 Using index0 the number of items from the end starting from 1 or 0 some truth operators like first, last then the length property which will be familiar if you are a js developer, and finally the parent which refers to the context outside the loop.
