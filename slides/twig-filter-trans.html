Translating Strings

Drupal 7


  <? print t('Hello @name', array('@name' => $name); ?>

Drupal 8


  Hello {{ name|t }}


  {% trans %} Hello {{ name }} {% endtrans %}
In Drupal 7 if you wanted to translate a string that included variables you needed to pass the values as an array using placeholder tokens. In Twig, you can just use the t filter, but in this case it's better to use a translation block, so that the entire string can be translated but variables will be escaped and printed correctly.
