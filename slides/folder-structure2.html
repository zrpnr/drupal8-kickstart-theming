Folder Structure (Core)


  ┣ core/
    ┃
    ┣ modules/
    ┃
    ┗ themes/ 
      ┃
      ┣ bartik
      ┣ classy
      ┣ seven
      ┣ stable
      ┣ stark
in the core folder you'll see a similar sub structure, inside themes are the core themes shipped with Drupal that you can use for reference. You'll likely use one of these as a "base" theme that your theme will inherit from. If you dont' specify a base your theme will inherit from "stable".
