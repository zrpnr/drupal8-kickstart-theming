CBEM Syntax

Component Element Modifier


  .component-name {}
  
  .component-name__element-name {}
  
  .component-name--modifier-name {}
  
  .component-name__element-name--modifier-name {}
BEM stands for Block Element Modifier, although in drupal the word "block" already carries a lot of meaning, so it is generally referred to as Component Element Modifier. This refers to a system for naming classes, allowing you to more consistently and meaningfully name the elements on your page and write more specific css with less inheritance, making it more modular and re-usable. everything is a class and nothing is nested.
