Creating accessible themes

https://www.drupal.org/node/464472

all the core themes use html5 elements, support responsive layouts (with the exception of stark), and have improvments for accessibility. Visual elements are improved for legibility to limited sight users and making content and forms legible to screen readers or assistive devices. Search engine form and presentation Drag and Drop functionality Color contrast and intensity Adding skip navigation to core themes Image handling Form labeling Web accessibility means that people with disabilities can perceive, understand, navigate, and interact with and contribute with the Web,
