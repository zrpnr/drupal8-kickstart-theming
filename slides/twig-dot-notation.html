Dot notation


  {# Print the value of the field without the field markup #}

  {{ node.field_name.value }}

Escaped by default


    {# Do it again, but like you don't care #}
  
    {{ node.field_name.value|raw }}
A dot can be used to access the attributes of a variable, or print a nested value. The dot syntax in twig works for nested array values, as well as object properties and methods. Notice the pipe there in that last example- that's using a feature of twig called "filtering".
